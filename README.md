# Long Hui — CV

Printable HTML CV. Open `index.html` in a browser or visit the GitHub Pages URL.

## Structure

- `index.html` — The CV (single file, no JS)
- `style.css` — Screen and print styles
- `docs/` — Source content and requirements

## Printing

Use **Cmd+P** (or Ctrl+P) → Print to PDF. The stylesheet handles B&W-friendly formatting, proper margins, and page-break control automatically.
